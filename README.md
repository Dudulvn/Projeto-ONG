## 3. Problema Identificado

A instituição **Arca Proteção**, que atua no resgate e tratamento de animais de rua, não possui um sistema digital para divulgar e gerenciar suas atividades.

Atualmente:

- As informações dos animais resgatados ficam registradas em planilhas ou documentos físicos.
- Não existe um site atualizado com os animais disponíveis para adoção.
- Interessados precisam entrar em contato por telefone ou presencialmente para obter informações.
- Não há controle organizado do histórico de adotantes.
- O acompanhamento do tratamento veterinário dos animais não é centralizado em um único sistema.

### Isso causa:

- Demora no processo de adoção.
- Dificuldade na divulgação dos animais resgatados.
- Falta de organização das informações.
- Risco de perda de dados importantes.
- Dificuldade para acompanhar o histórico de resgates e tratamentos.

---

## 4. Solução Proposta

Será desenvolvido um **sistema web utilizando Python**, com integração a banco de dados, para:

- Cadastrar animais resgatados.
- Registrar informações sobre tratamento veterinário.
- Disponibilizar animais para adoção em um site atualizado.
- Cadastrar e organizar dados de adotantes.
- Centralizar todas as informações em um único sistema digital.

O sistema tornará o processo mais organizado, seguro e eficiente.
